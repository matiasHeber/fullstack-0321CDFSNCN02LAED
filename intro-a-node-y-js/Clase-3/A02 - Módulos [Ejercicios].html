NPM y Requires

Repasemos un poco lo visto en el material virtual creando un proyecto nuevo de Node e implementando paquetes con NPM.

Paquete externo:

  1. Crea una carpeta vacía llamada pruebaNpm.
  2. Abre la terminal o consola y situate en la carpeta creada.
  3. Inicia un proyecto de node desde la terminal.
  4. Luego configura el proyecto con la información necesaria.
  5. Abre el proyecto en Visual Studio Code y modifica el código para que el archivo principal sea app.js.
  6. Crea el archivo app.js.
  7. Desde la terminal instala el paquete de node popular-movie-quotes. Recuerda setear la dependencia para que se vea reflejada en el archivo package.json.
  8. En el archivo app.js declara una variable que requiera el paquete instalado para poder utilizarlo.
  9. Imprime en consola el método getRandomQuote() para obtener al menos tres citas de una película aleatoria.
  10.  Desde la consola ejecuta tu archivo app.js para ver los resultados.

Paquete local:

  1. Crea una carpeta dentro del proyecto llamada autos.
  2. Dentro de la carpeta crea el archivo index.js.
  3. En este archivo deberás crear una variable autos que tenga un objeto como valor.
  4. La primera propiedad deberá ser un lista de todos los autos en formato de array.
  5. Dentro del array habrá muchos objetos (uno por cada auto).
  6. Los objetos deberán tener las siguientes propiedades:
    →  marca -> String con la marca del auto
    →  modelo -> String con el modelo del auto
    →  anio -> Number con el año de fabricación del auto
    →  color -> String con el color del auto

→  describirse -> Método que retorna un string incluyendo todas las características del auto.

  1. Crea objetos de autos de al menos 3 marcas con 3 modelos por cada una
  2. Exporta la variable que contiene el array de objetos.
  3. Utiliza desde app.js la variable exportada e imprimí en consola la lista de todos los autos para ver que esté funcionando correctamente.
  4. Desde app.js, recorré el array de resultado y llama al método describirse de cada auto.
  5. Al objeto autos agregale los siguientes métodos:
    →  porColor -> Debe devolver una colección de autos dependiendo el color pasado como parámetro. En caso de no tener resultados deberá devolver un mensaje para avisar la falta de estos. Para esto, deberás dentro del método:

  1. Definir una variable de resultados
  2. Realizar un for que recorra todo los autos
  3. Dentro del for chequear la condición de color con un if
  4. Dentro del if (si se cumple la condición) agregar el auto a los resultados
  5. Retornar la variable resultados.


→ porMarca -> Debe devolver la colección de autos dependiendo la marca que se pase como parámetro. En caso de no tener resultados deberá devolver un mensaje para avisar la falta de estos.
→ porAnio -> Debe devolver una colección de autos dependiendo el año que se pase como parámetro. En caso de no tener resultados deberá devolver un mensaje para avisar la falta de estos.

Invoca los distintos métodos con diversos parámetros.
